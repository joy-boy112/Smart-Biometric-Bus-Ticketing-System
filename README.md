# Smart Biometric Bus Ticketing System 🚍

An ESP32-based smart bus ticketing system using RFID authentication, keypad input, WiFi connectivity, and a web-based mobile ticket interface.

## Features
- RFID card ticket generation
- Smart fare calculation
- Real-time web ticket display
- Database ticket storage
- Keypad-based bus stop selection
- WiFi + ESP32 integration
- Mobile-friendly ticket UI

## Technologies Used
- ESP32
- Arduino IDE
- RFID RC522
- Keypad
- WiFi
- HTML/CSS
- WebServer
- HTTPClient

## Components
- ESP32
- RC522 RFID Module
- 4x4 Keypad
- WiFi Network

## Project Idea
This project aims to modernize traditional bus ticket systems by enabling digital ticket generation and smart fare management using embedded systems and IoT concepts.
